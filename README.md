# ⏰ Theme Clock

A modern and visually appealing **Theme Clock Web App** that displays real-time time with dynamic theme switching (Light/Dark mode). Built using **HTML, CSS, and JavaScript**.

---

## 🚀 Live Demo

🔗 https://siddh-jain01.github.io/THEME_CLOCK/

## 📂 Source Code

🔗 https://github.com/siddh-jain01/THEME_CLOCK

---

## ✨ Features

* 🕒 Real-time clock display
* 🌗 Light & Dark theme toggle
* 🎨 Smooth UI and modern design
* ⚡ Fast and lightweight
* 📱 Responsive design (works on mobile & desktop)

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling & themes
* **JavaScript (ES6)** – Functionality

## ⚙️ How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/siddh-jain01/THEME_CLOCK.git
```

2. Open the project folder

```bash
cd THEME_CLOCK
```

3. Run the project

* Open `index.html` in your browser
